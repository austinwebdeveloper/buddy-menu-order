=====================================
How to install Foodpress Extensions
=====================================
Foodpress extensions can be installed just like a wordpress plugin.

Step 1:
Go to WP Admin > Plugins > Add New

Step 2: 
Select "Upload" and choose the zip file of the extension (ONLY) 

Step 3:
Click Install Now and follow the screen to activate.


-------

You can also install this foodpress Extension by upzipping the file content and uploading the content to "wp-content/plugins/" directory.
